# Spacer

<ImageContainer withBorder={true}>
  ![Example spacer](/images/ui-components/spacer.png)
</ImageContainer>

A link button has the following properties:

- `spacerSize`: the amount of space the component should take up. One of `XS`, `S`, `M`, `L`, `XL`.

For example:

<Tabs items={[ 'Typescript SDK', 'GraphQL']}>
    <Tab>
        <UISpacerTypescriptSDK />
    </Tab>

    <Tab>
        <UISpacerGraphQL />
    </Tab>

</Tabs>
